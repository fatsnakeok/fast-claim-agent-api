sequenceDiagram
    actor User as 用户
    participant Ctrl as InsuranceController
/api/insurance/underwrite
    participant AS as AgentService
    participant Platform as Embabel Platform
    participant UA as UnderwritingAgent
@Agent(UTILITY)
    participant LLM as DeepSeek LLM
(fast)
    participant DS as DataService
    participant RS as RiskCalculationService
    participant PS as PremiumCalculationService
    participant QR as QuoteRepository
    participant DB as H2 Database

    User->>Ctrl: POST /api/insurance/underwrite
{userId, userInput}
    activate Ctrl
    Ctrl->>Ctrl: @PreAuthorize 权限校验
hasAuthority('underwriting:write')
    Ctrl->>AS: processUnderwriting(userId, userInput)
    activate AS

    AS->>Platform: findAgent(UnderwritingAgent.class)
    Platform-->>AS: agent 实例

    AS->>Platform: createAgentProcessFrom(agent, options, input)
    Platform-->>AS: AgentProcess

    AS->>AS: process.blackboard.bind("userId", userId)
    AS->>AS: CompletableFuture.supplyAsync(process::run)
ThreadPoolExecutor(2,4,60s,Queue100)

    Note over AS,UA: === Agent 流水线开始 (PlannerType.UTILITY) ===

    AS->>UA: [1] extractVehicleInfo(input, context)
    activate UA
    UA->>UA: 从 Blackboard 读取 userId
    UA->>DS: findCustomer(userId) 前置检查
    DS->>DB: SELECT * FROM customer WHERE user_id=?
    DB-->>DS: Customer 实体
    DS-->>UA: Customer (非 sentinel)
    UA->>LLM: generateText(提取车辆信息 prompt)
    LLM-->>UA: JSON {"brand":"Toyota","model":"RAV4","licensePlate":"LOW001"}
    UA->>UA: 反序列化为 VehicleInfo
    UA-->>AS: VehicleInfo
    deactivate UA

    AS->>UA: [2] lookupCustomer(input, context)
    activate UA
    UA->>DS: findCustomer(userId)
    DS->>DB: SELECT * FROM customer WHERE user_id=?
    DB-->>DS: Customer 实体
    DS-->>UA: Customer
    UA-->>AS: Customer
    deactivate UA

    AS->>UA: [3] lookupVehicle(vehicleInfo, customer, context)
    activate UA
    UA->>DS: findVehicle(info, customer)
    activate DS
    DS->>DB: 1) licensePlate 精确匹配
SELECT * FROM vehicle WHERE license_plate=?
    DB-->>DS: Vehicle + 归属校验
    alt 车牌未命中
        DS->>DB: 2) brand + model 匹配
SELECT * FROM vehicle WHERE customer_id=?
        DB-->>DS: 客户名下车辆列表, 逐一匹配品牌型号
        alt 品牌型号未命中
            DS->>DS: 3) 客户名下唯一车辆?
size==1 → 返回唯一车辆
        end
    end
    DS-->>UA: Vehicle / Vehicle.lookupFailed()
    deactivate DS
    UA-->>AS: Vehicle
    deactivate UA

    AS->>UA: [4] assessRisk(customer, vehicle, context)
    activate UA
    UA->>UA: sentinel 检查 (Customer/Vehicle)
    UA->>RS: calculate(customer, vehicle)
    activate RS
    RS->>RS: 6因子评分:
年龄 + 驾龄 + 事故×15
+ 车龄 + 车辆价值
→ 钳制[0,100]
    RS-->>UA: riskScore (double)
    deactivate RS

    UA->>PS: calculate(vehicleValue, riskScore, "COMPREHENSIVE")
    activate PS
    PS->>PS: 车价×2%×风险系数×险种系数
BigDecimal.setScale(2, HALF_UP)
    PS-->>UA: premium (double)
    deactivate PS

    alt riskScore ≤ 60
        UA-->>AS: LowRiskQuote(customer, vehicle, riskScore, premium)
        Note over UA: @State 路由 → LowRiskQuote.execute()
        UA->>QR: save(Quote{status=APPROVED})
        QR->>DB: INSERT INTO quote
        DB-->>QR: quoteId
        QR-->>UA: Quote

    else 60 < riskScore < 80
        UA-->>AS: MediumRiskReview(customer, vehicle, riskScore, premium)
        Note over UA: @State 路由 → MediumRiskReview.execute()
        UA->>QR: save(Quote{status=REFERRED, expiresAt=+30d})
        QR->>DB: INSERT INTO quote
        DB-->>QR: quoteId
        QR-->>UA: Quote

    else riskScore ≥ 80
        UA-->>AS: HighRiskDecline(customer, vehicle, riskScore, premium)
        Note over UA: @State 路由 → HighRiskDecline.execute()
        UA->>QR: save(Quote{status=DECLINED, rejectionReason})
        QR->>DB: INSERT INTO quote
        DB-->>QR: quoteId
        QR-->>UA: Quote
    end
    deactivate UA

    Note over AS,UA: === Agent 流水线结束 (@AchievesGoal) ===

    AS->>AS: processResult.last(UnderwritingResult.class)
    AS-->>Ctrl: UnderwritingResult{status, riskScore, premium, message}
    deactivate AS

    alt status == APPROVED / REFERRED / DECLINED
        Ctrl-->>User: 200 OK + UnderwritingResult JSON
    else status == ERROR
        Ctrl-->>User: 422 Unprocessable Entity + UnderwritingResult JSON
    end
    deactivate Ctrl

保单查询 US-U4 
sequenceDiagram
    actor User as 用户
    participant Ctrl as InsuranceController
    participant PolSvc as PolicyService
    participant PR as PolicyRepository
    participant DB as H2 Database

    rect rgb(20, 40, 60)
        Note over User,DB: 场景A: 按 userId 查询保单列表
        User->>Ctrl: GET /api/insurance/policies?userId=low-risk-user
        activate Ctrl
        Ctrl->>Ctrl: @PreAuthorize hasAuthority('policies:read')
        Ctrl->>PolSvc: findByUserId("low-risk-user")
        activate PolSvc
        PolSvc->>PR: findByCustomer_UserId("low-risk-user")
        PR->>DB: SELECT p.* FROM policy p
JOIN customer c ON p.customer_id=c.id
WHERE c.user_id='low-risk-user'
        DB-->>PR: List<Policy>
        PR-->>PolSvc: List<Policy> (or empty list)
        PolSvc-->>Ctrl: List<Policy>
        deactivate PolSvc
        Ctrl->>Ctrl: stream().map(PolicyResponse::from)
        Ctrl-->>User: 200 OK + List<PolicyResponse>
        deactivate Ctrl
    end

    rect rgb(20, 60, 40)
        Note over User,DB: 场景B: 按 policyNumber 查询保单详情
        User->>Ctrl: GET /api/insurance/policies/POL-170001-ABC123
        activate Ctrl
        Ctrl->>Ctrl: @PreAuthorize hasAuthority('policies:read')
        Ctrl->>PolSvc: findByPolicyNumber("POL-170001-ABC123")
        activate PolSvc
        PolSvc->>PR: findByPolicyNumber("POL-170001-ABC123")
        PR->>DB: SELECT * FROM policy WHERE policy_number=?
        DB-->>PR: Optional<Policy>
        PR-->>PolSvc: Optional<Policy>
        alt 保单存在
            PolSvc-->>Ctrl: Policy
            Ctrl->>Ctrl: PolicyResponse.from(policy)
            Ctrl-->>User: 200 OK + PolicyResponse
        else 保单不存在
            PolSvc-->>Ctrl: throw BizException("保单不存在")
            Ctrl->>Ctrl: GlobalExceptionHandler → 404
            Ctrl-->>User: 404 Not Found
        end
        deactivate PolSvc
        deactivate Ctrl
    end
超时保护与 Blackboard 错误回退 120s 超时 · StuckHandler 
sequenceDiagram
    participant AS as AgentService
    participant Future as CompletableFuture
    participant UA as UnderwritingAgent
    participant Stuck as StuckHandler

    AS->>AS: ThreadPoolExecutor(2,4,60s,Queue100)
    AS->>Future: supplyAsync(process::run)
    activate Future

    alt 正常完成 (≤120s)
        Future-->>AS: ProcessResult (completed)
        AS->>AS: completed.last(UnderwritingResult.class)
        AS->>AS: return result

    else LLM 调用超时 / Agent 卡住
        Note over Future: future.get(120, SECONDS)
        Future-->>AS: TimeoutException
        AS->>Future: cancel(true)
        AS->>UA: handleStuck(agentProcess)
        activate UA
        UA->>UA: log.warn("超时 STUCK — Blackboard entries")
        UA-->>AS: StuckHandlerResult(NO_RESOLUTION)
        deactivate UA

        AS->>AS: 从 Blackboard 回退读取
process.blackboard.get("underwriting_error")
        AS->>AS: 包装为 UnderwritingResult.error()
        Note over AS: 返回 ERROR 而非抛出异常，
保证 Controller 总能拿到响应体
    end
    deactivate Future
